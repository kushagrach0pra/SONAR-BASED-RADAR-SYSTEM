🛰️ Arduino-Based Radar (Sonar) System

What if your Arduino could "see" its surroundings?

That question inspired me to build an Arduino-based Radar (Sonar) System capable of detecting obstacles in real time.

Components:
Arduino Uno
1) HC-Sr04 Ultrasonic Sensor
2) 2 - LED BULBS
3) 2 - 220ohms resistors
4) 1- Sg 90 servo motor
5)A 16x2 LCD DISPLAY WITH I2C module
6) A buzzer
7) USB CABLE AND SOME JUMPER WIRES

Although I know that my project is not tidy enough but i am still learning each day and will definitely succeed one day.


This project strengthened my skills in embedded systems, Arduino programming, sensor interfacing, hardware integration, circuit debugging, and real-time data processing. It also taught me that successful engineering is not just about writing code—it's about solving problems through persistence, testing, and continuous improvement.
